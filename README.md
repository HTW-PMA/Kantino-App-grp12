# Mensa App Berlin

Eine React Native App für Berliner Mensen und Cafeterias mit Speiseplan-Anzeige, Favoriten-Verwaltung und Navigation.

## Features

* Übersicht aller Berliner Mensen und Cafeterias
* Aktuelle Speisepläne anzeigen
* Lieblings-Mensen speichern
* Navigation zu Mensen via Google Maps
* AI-Chatbot für Fragen rund um Mensen und Speisen
* Persistente Datenspeicherung (AsyncStorage, App-Bundle)

## Installation

1. **Dependencies installieren**
   ```bash
   npm install
   ```

2. **App starten**
   ```bash
   npx expo start --clear
   ```

3. **App öffnen**
   - Expo Go App auf dem Smartphone
   - Android Emulator
   - iOS Simulator
